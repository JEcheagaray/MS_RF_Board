Detailed Description

Battery Monitoring Software Component (SWC) for MS_RF_Board.

This module monitors the battery voltage of the MS_RF_Board, which uses three 18650 batteries in series. It calculates the state of charge (SOC) based on the debounced voltage readings and logs the battery status.
