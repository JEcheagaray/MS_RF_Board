Detailed Description

Header file for Current Sensing Software Component (SWC) for MS_RF_Board.

This header provides the declarations for monitoring current from two sensors, SENSOR_1 and SENSOR_2, and interfaces to retrieve raw and debounced measurements. It also defines a cyclic process to ensure regular updates of current readings.
