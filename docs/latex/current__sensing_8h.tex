\doxysubsection{Detailed Description}
Header file for Current Sensing Software Component (SWC) for MS\+\_\+\+RF\+\_\+\+Board. 

This header provides the declarations for monitoring current from two sensors, SENSOR\+\_\+1 and SENSOR\+\_\+2, and interfaces to retrieve raw and debounced measurements. It also defines a cyclic process to ensure regular updates of current readings.
